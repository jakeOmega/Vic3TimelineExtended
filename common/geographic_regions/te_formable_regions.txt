# Geographic regions used by the new formable countries in
# common/country_formation/te_formable_countries.txt.
#
# Vic3 quirk: a formable's `geographic_region = X` field only enumerates
# states from X's `state_regions = { ... }` block. `strategic_regions = { ... }`
# is silently ignored in that code path (it works fine for
# `is_in_geographic_region = X` triggers — different code path). So vanilla
# `geographic_region_europe` / `geographic_region_africa` (strategic_regions
# form) cannot drive a formable's required-states list, even though they're
# semantically what we want. We define explicit-state-list mirrors instead:
#
# - geographic_region_united_europe       (mirrors vanilla geographic_region_europe)
# - geographic_region_united_africa       (mirrors vanilla geographic_region_africa)
# - geographic_region_united_north_america
# - geographic_region_united_earth        (every land strategic region)
#
# Those four are auto-generated into te_formable_regions_generated.txt by
# scripts/generators/gen_formable_regions.py — re-run after any vanilla
# strategic-region rebalance.
#
# Per-formable region pointers:
#   AFU -> geographic_region_united_africa (mod, generated)
#   EUN -> geographic_region_united_europe (mod, generated)
#   UNA -> geographic_region_united_north_america (mod, generated)
#   UNE -> geographic_region_united_earth (mod, generated)
#   INM -> geographic_region_intermarium (hand-maintained, below)
#   EAF -> geographic_region_east_african_federation (hand-maintained, below)
#         — also referenced by `is_major_east_african_power` in
#         common/scripted_triggers/formable_gating_triggers.txt.
#   DAR -> geographic_region_dar_al_islam (hand-maintained, below)
#         — also referenced by `is_major_dar_al_islam_power` in
#         common/scripted_triggers/formable_gating_triggers.txt.
#   BHT -> vanilla geographic_region_india (state_regions form, works as-is)
#   BHA -> use_culture_states (no region needed)
#   IDN/NUS -> explicit state lists in country_formation (no region needed)
