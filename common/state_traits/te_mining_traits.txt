# State traits for the mod's custom mineable resources (GH #110).
#
# Mod custom mines (defined in common/buildings/extra_buildings.txt) need
# regional quality/ease-of-mining bonuses analogous to vanilla traits like
# state_trait_bergslagen (iron+lead) or state_trait_appalachian_coal. Deposit
# size is already modelled by amounts in deposits_config.json; these traits
# represent quality.
#
# Throughput modifiers must be registered in
# common/modifier_type_definitions/mod_entity_modifier_types.txt.
#
# Trait references are attached to states via state_trait_config.json,
# injected by resources.py into map_data/state_regions/*.txt on every regen.
#
# INJECT: directives extend existing vanilla state traits (e.g. atacama_desert,
# bone_valley) so the in-game state encyclopedia shows one geographic trait
# per region instead of stacking a vanilla + mod trait on the same feature.
