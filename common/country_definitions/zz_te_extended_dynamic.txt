# Vanilla `99_dynamic.txt` only defines D00–D99, which caps create_dynamic_country
# at 100 simultaneous dynamic countries. In deep timeline saves the pool fills up
# (decolonization, civil wars, revolutions, etc.), after which further dynamic creates
# silently fail with no error logged. The 900 extra slots below give us headroom.
# Loaded after 99_dynamic.txt via the zz_ prefix.
