# =============================================================================
# Vic3TimelineExtended — modern (post-vanilla) ship types
# =============================================================================
# These ships extend vanilla 1.13's ship roster into the mod's late-era timeline.
# Each ship is loosely modeled on a vanilla parent type and reuses that parent's
# modification slots (ship_mod_<parent>_*) so we don't duplicate every armor /
# guns / propulsion / range tier.
#
# Era / tech mapping:
#   era 6  sonar                        → asw_destroyer
#   era 7  advanced_submarine_technology → nuclear_submarine
#   era 7  guided_missiles              → guided_missile_ship
#   era 7  nuclear_energy               → nuclear_supercarrier
#   era 8  precision_guided_munitions   → arsenal_ship
#   era 9  rapid_deployment_forces      → expeditionary_sea_base
#   era 10 hypersonic_weapons           → hypersonic_missile_platform
#   era 11 directed_energy_weapons      → pulsed_laser_escort
#   era 11 swarm_technology             → swarm_coordination_vessel
# =============================================================================
