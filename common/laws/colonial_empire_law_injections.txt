# Colonial Empire — Vanilla Law Injections
# ============================================================================
# Adds the four colonial-empire modifier types to vanilla laws:
#
#   country_colonial_stability_drift_add        — read by the bar always
#   country_colonial_garrison_effectiveness_add — read when Garrison is active
#   country_colonial_invest_effectiveness_add   — read when Invest is active
#   country_colonial_assim_effectiveness_add    — read when Assimilation is active
#
# Magnitudes match the corresponding IF-block values that previously lived
# in common/scripted_progress_bars/extra_progress_bars.txt. The bar reads
# aggregated totals via owner.modifier:<name>; the policy-button tooltips
# preview the effectiveness aggregate via GetValueWithBreakdownFor so the
# player sees what their laws contribute BEFORE pressing the button.
#
# Mirrors the sol_expectations_vanilla_injections.txt INJECT pattern: each
# block extends an existing vanilla law non-destructively.

# === Minority Rights / Citizenship: handled via DIRECT EDITS, not INJECT ===
# These laws are redeclared by the mod (extra_laws.txt for minority_rights,
# modified.txt for the citizenship variants). INJECT directives don't merge
# into mod redeclarations -- the latter wins silently. The colonial-stability
# contributions for these laws live alongside their other modifiers in
# common/laws/extra_laws.txt and common/laws/modified.txt.

# law_neocolonialism is a mod-only law (no vanilla def). Direct edit in
# common/laws/extra_laws.txt.
