# ======================================================================
#  LAW ENACTMENT EVENTS – ON ACTIONS
#  Wires extra_law_events and ministry_law_events into the three
#  law-enactment checkpoint on-actions (advance / debate / stall).
#
#  Classification (by narrative context, not option outcome -
#  ALL event options advance the law; the difference is tradeoffs):
#    ADVANCE  – momentum narrative (crises, scandals, rallies that
#               naturally fit when the law is already progressing)
#    DEBATE   – balanced narrative (genuine two-sided arguments)
#    STALL    – opposition narrative (pushback events that fit when
#               the law faces resistance; options still advance)
#
#  Some events appear in two categories where the narrative fits both
#  (e.g. an opposition-IG event with debate-style options can sit in
#  both DEBATE and STALL).
#
#  Compromise/amendment events live in DEBATE only — their narratives
#  are about deal-making, not opposition pushback.
# ======================================================================
