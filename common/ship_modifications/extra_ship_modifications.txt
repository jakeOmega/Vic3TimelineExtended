# =============================================================================
# Vic3TimelineExtended — modern ship modifications
# =============================================================================
# Replaces the vanilla parent-ship slot mods (e.g. ship_mod_super_dreadnought_*)
# that the modern ships were using. Each modern ship now has its own
# thematically-modern slot mods. Modifier values match the vanilla parent's
# (preserves balance); construction goods are modernized; names + flavor are
# updated to fit the era.
#
# Tier scaling matches vanilla pattern: light/medium/high are scalar steps on
# the same modifier axes. Some tiers add a small extra effect where the theme
# justifies (e.g. heavy carrier propulsion adding small blockade strength).
# =============================================================================
