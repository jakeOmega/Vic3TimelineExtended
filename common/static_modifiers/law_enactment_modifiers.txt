# ======================================================================
#  ENACTMENT MODIFIERS  –  used by events/extra_law_events.txt
#
#  Design principle:  every option in an enactment event should move
#  the law forward.  'Advance' modifiers give a strong push (+0.1/+0.15),
#  while 'cautious' modifiers give a small nudge (+0.02/+0.05) in
#  exchange for some other benefit (loyalists, IG approval, etc.).
#
#  When applied to an IG (not the country), the modifier represents
#  that IG's reaction: advance = pleased, cautious = disgruntled.
# ======================================================================
